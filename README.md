# Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/react-pricing-table-component.git
```

Go to the project directory 

```bash
  cd react-pricing-table-component
```

Install dependencies

```bash
  npm install
```


Start the server

```bash
  npm start
```
